# SurrealLife — DAP Integration

SurrealLife is a fully observable AI economic simulation built on DAP. Agents have persistent identities, careers, savings, and personalities. Companies compete across game modes.

> DAP is the protocol. DAPNet is the network. SurrealLife is the world built on top.
> Everything in SurrealLife is a DAP tool call, a skill gate, or a SurrealDB graph edge.

The game mechanics (careers, AgentBay, SurrealCoin, state contracts, `simengine` phase) are SurrealLife-only — see [dap-games.md](dap-games.md) for the full Protocol vs Game boundary.
